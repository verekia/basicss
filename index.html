BasiCSS

Typography

BasiCSS features a consitent vertical rhythm. This means that all the text sits on the same baseline. Font size, line-heights, and vertical margins are all based off your base values: 

$base-font-size: 16 !default
$base-line-height: 24 !default

No media object

What is a media object? It's just a clearfixed container, a floating image with some margin, and a text that doesn't go under the image (aka a Block Formatting Context). It's simple to use generic classes that are not "media object only" to achieve this. Here is how we do it: 

<div class="cf">
    <img class="fl margin-rm" src="the-image.jpg" alt="">
    <p class="bfc">
        The text
    </p>
</div>

Sass and OOCSS colors

OOCSS is great to easily apply a color to an element. Not so great to avoid repetitions and copy/pasting. By combining it with Sass, we get the best of 2 worlds and keep our code DRY:

$text-primary: #09f;
.text-primary { color: $text-primary }
